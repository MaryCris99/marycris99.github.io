JavaScript Cheat Sheet

JavaScript is a programming language that powers the dynamic behavior on most websites. Alongside with HTM and CSS, it is a core technology that makes the web run.

Variables

Keyword Description Example
var Declares a variable with optional initialization var x = 5;
let Declares a block-scoped variable with optional initialization let y = "Hello, world!";
const Declares a block-scoped constant with required initialization const PI = 3.14159;

Operators

Operator Description Example
+ Addition 1 + 2
- Subtraction 3 - 2
* Multiplication 2 * 3
/ Division 6 / 2
% Modulus (remainder) 7 % 3

Functions

Keyword Description Example
myFunction() Declare a function: function myFunction() { // function body }

Call a function:

myFunction();

Arrays

Keyword Description Example
let myArray = [1, 2, 3]; Access an array element: myArray[0];

Console.log()

Keyword Description Example
console.log The console.log() method is used to log or print messages to the console.It can also be used to print objetcts and other info. console.log('Hello World!'); // Prints: Hello World!

Multi-line Comments

Keyword Description Example
/* */ In JavaScript, Multi-line comments are created by surrounding the lines with /* at the beginning and */ at the end. It helps for indication some hints or explaining a block of code. /*Please put your code here*/

Booleans

Keyword Description Example
let x=true; Booleans are a primitive data type. They can be either true or false. let noClass=true;

Built-in Objects

Keyword Description Example
Built-in objects contain methods that can be called by appending the object name with a period., the method name, and a set of parenthesis. Math.random() //Math is the Built-in object.
